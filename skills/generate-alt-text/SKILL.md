# Generate Alt Text

## Rules

- All text visible on the image (for screenshots etc.) must be exactly included.
- A short note describing the nature of the image itself should go first.
- For charts or data visualizations, highlight the most interesting numbers.
- Keep it concise but complete enough to convey the image's key meaning.
- Present alt text on a single line so it can be used directly in Markdown image syntax.
